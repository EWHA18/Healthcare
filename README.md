# 🏷️#건강기능식품 프로젝트

## ✏ Features
    1) 사용자 직접 입력 (단일 사용자)
    2) CSV 파일 입력 (단일/다수 사용자)

## ✏ Initialization
- clone the Repository
```
$ git clone https://github.com/EWHA18/Healthcare.git
```
<br/>

- Install npm packages & Start React
```
$ cd healthcare
$ npm i
$ npm start
```
<br/>

- Start Flask
```
$ cd backend
$ flask run
```
